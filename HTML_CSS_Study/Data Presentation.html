Ordered Lists

the lists can be ordered by numbers or letters: the '< ol >' tag is used to create the list and each item on the list is indicated by the '< li >' tag>:
  1. HTML
  2. CSS
  3. JavaScript
we can pass a “type” attribute to the '< ol >' tag to indicate the type of order:
  1. type="1" - numbers;
  2. type="A" - uppercase alphabet;
  3. type="a" - lowercase alphabet;
  4. type="I" - uppercase Roman numbers
  5. type="i" - lowercase Roman numbers

Unordered Lists

similar to Ordered Lists, but start with '< ul >':
  • HTML
  • CSS
  • JavaScript

Description Lists

These lists are formed by the '< dl >' tag, where we set the name of a term ('< dt >') and then we list descriptions about that term ('< dd >')
HTML
- HyperText Markup Language
CSS
- Cascading Style Sheet

Progress and Meter

to use the '< progress >' tag, we pass the total progress value in the “max” attribute, and the progress value in “value”
it is possible to use the '< meter >' tag, indicating which values are considered high and low with the high and low attributes
